# Pharma Industry Login System (Spring Boot)

This project is a simple login system for the Pharma Industry, built with Java, Spring Boot, and Thymeleaf. It features a modern login page and a welcome message upon successful login. No database is required; credentials are hardcoded for demonstration purposes.

## Features
- Login page with username and password fields
- Modern UI using Thymeleaf and Bootstrap
- Welcome page after successful login
- Standard Spring Boot project structure

## Getting Started

### Prerequisites
- Java 17 or later
- Maven 3.6+

### Running the Application
1. Navigate to the project directory:
   ```sh
   cd pharma-login
   ```
2. Build and run the application:
   ```sh
   mvn spring-boot:run
   ```
3. Open your browser and go to [http://localhost:8080](http://localhost:8080)

### Default Credentials
- **Username:** admin
- **Password:** password

## Project Structure
- `src/main/java` - Java source code
- `src/main/resources/templates` - Thymeleaf templates
- `src/main/resources/static` - Static assets (CSS, JS)
